# KoeMo（コエモ）

> あなたの声に表情がつくギミック

KoeMo（コエモ）は、あなたの声に表情がつくギミックです。マイクに入力された声をリアルタイムで分析し、感情（喜び・悲しみ・怒り・平静 など）を推定して、VRChatアバターの表情に自動で反映します。

「楽しく話しているのに、アバターの表情はずっと無表情……」
そんなもどかしさを、声から感情を読み取り、OSCを通じてリアルタイムでアバターに伝えることで解決します。

## 特徴

- マイクの声をリアルタイム解析
- 喜び・悲しみ・怒り・平静などの感情を自動推定
- すべてローカル処理。音声データの外部送信・保存は一切なし
- キャリブレーション機能で自分の声に合わせて精度調整
- OSC経由でVRChatアバターの表情にリアルタイム反映
