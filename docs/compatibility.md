## 動作確認に協力する

ご自身のアバターやギミックでKoeMoの動作確認をしていただいた場合、ぜひ情報をお寄せください。

- [GitHub Issues](https://github.com/sanbongazin/VoiceEmo/issues) に報告
- 確認状況をこのページに追加します

## 確認状況の見方

| マーク | 意味 |
|---|---|
| ✅ 動作確認済み | 問題なく動作することを確認 |
| ⚠️ 一部制限あり | 動作するが注意点や制限がある |
| ❌ 非対応 | 現時点では動作しない |
| 🔍 確認中 | 現在テスト中 |
