# セットアップ

## インストール手順

1. [BOOTH](https://akihoshidatainc.booth.pm/items/8350327) で VoxEmo を購入し、`VoxEmo_Launcher.exe` をダウンロードします
2. 任意のフォルダに配置して起動します
3. 初回起動時にセットアップウィザードが自動で実行されます
   - 必要なファイルのダウンロード
   - 依存ライブラリのインストール
   - 初期設定の完了

コマンド操作やPython知識は一切不要です。ウィザードの指示に従うだけでセットアップが完了します。

## VRChatのOSC設定

VoxEmoはOSCを使ってVRChatと通信します。VRChat側でOSCを有効にしてください。

1. VRChatを起動する
2. メニュー → **OSC** → **Enable** をオンにする

## 次のステップ

セットアップが完了したら [キャリブレーション](/guide/calibration) を行いましょう。
