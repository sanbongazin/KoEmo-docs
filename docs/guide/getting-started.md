# はじめに

VoxEmo（コエモ）は、マイクに入力された声をリアルタイムで分析し、感情をVRChatアバターの表情に自動で反映するギミックです。

## VoxEmo でできること

- 声から「喜び・悲しみ・怒り・平静」などの感情を推定
- 推定結果をOSCでVRChatに送信し、アバターの表情を自動制御
- キャリブレーションで自分の声の特徴に合わせた調整
- 完全ローカル処理（音声データの外部送信なし）

## 動作の流れ

```
マイク入力 → VoxEmo（感情推定） → OSC送信 → VRChat（アバター表情変化）
```

## 必要なもの

- VRChatアカウントとOSC対応アバター
- Windows PC（詳細は[動作要件](/reference/requirements)を参照）
