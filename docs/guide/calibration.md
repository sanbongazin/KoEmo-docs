# キャリブレーション

キャリブレーションは、KoeMoを自分の声の特徴に合わせて調整する機能です。声の高さ・大きさ・話し方の癖は人によって大きく異なるため、キャリブレーションを行うことで感情推定の精度が向上します。

## キャリブレーションの手順

1. KoeMoのメイン画面で **「キャリブレーション」** ボタンをクリック
2. 画面の指示に従って、各感情（平静・喜び・悲しみ・怒りなど）を声に出して話す
3. 録音が完了すると自動でモデルを調整します
4. 完了後、感情推定を開始できます

## キャリブレーションのコツ

- **静かな環境**で行う（背景ノイズが少ない場所が理想）
- 各感情はなるべく**自然に**発声する（大げさにならなくてよい）
- 普段VRChatで話すときと**同じマイク・同じ音量設定**で行う

## キャリブレーションをやり直したい場合

メイン画面のキャリブレーションボタンから、いつでも再キャリブレーションできます。
